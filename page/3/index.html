自动登录 服务器部署

ssh不输入密码，自动登陆

1.在本机 ~/.ssh 下新建config ，添加服务器ip，以及host简称，格式如下：

Host emall
  HostName 139.129.128.51
  Port 22
  User root

保存后 ssh emall就相当于 ssh root@139.129.128.51
